# 📦 Project Name

A simple website application using Jikan API to replicate MyAnimeList. 

---

## ✨ Features

* Fast Node.js server
* Template rendering with EJS
* Automatic server restart using Nodemon
* Runs locally on `localhost:3000`

---

## 🛠 Tech Stack

* Node.js
* Express.js
* EJS
* Nodemon (for development)

## 🚀 Getting Started

Follow the steps below to run this project on your local machine.

### 1. Clone the Repository

```
git clone https://github.com/PwrSat/myAnimeList-Wannabe.git
cd your-repository
```

### 2. Install Dependencies

Open your terminal in the project folder and run:

```
npm install
```

### 3. Run the Application

Start the server with Node.js:

```
node index.js
```

Or use Nodemon for easier development:

```
nodemon index.js
```

### 4. Open in Browser

After the server is running, open the following URL in your browser:

```
http://localhost:3000/
```

---

## ⚠️ Requirements

Make sure the following tools are installed on your device:

* Node.js (recommended version 16 or higher)
* npm (comes with Node.js)

Check if they are installed:

```
node -v
npm -v
```

---

## 📦 Install Nodemon (Optional)

Nodemon will automatically restart the server when you make changes to the code.

Install it globally:

```
npm install -g nodemon
```

Then run the project with:

```
nodemon index.js
```
